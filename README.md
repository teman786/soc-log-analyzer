# 🛡️ SOC Log Analyzer (Python GUI Tool)

A simple Python-based SOC (Security Operations Center) tool that analyzes log files to detect brute force attacks based on failed login attempts.

## 🚀 Features
- Import log file using GUI
- Detect failed login attempts
- Identify suspicious IP addresses
- Generate alerts based on threshold
- Simple and user-friendly interface

---

## 🧠 How It Works
- Reads log file line by line
- Extracts IP addresses using regex
- Counts failed login attempts per IP
- Flags IPs exceeding threshold (>=3 attempts)
